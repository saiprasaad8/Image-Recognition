# Image-Recognition
Vision AI- Image Recognition

## Requirements
- Python 3.8+
- TensorFlow 2.x
- Matplotlib, Seaborn, scikit-learn
- Flask (for demo app)

`pip install -r requirements.txt`

## Running the Code

- Open the Google Colab file
- Run the cells
- The final model will be saved as transfer_model.h5 for deployment
